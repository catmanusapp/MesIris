🚀 Déploiement & Mises à jour

Publiez l'application sur une URL publique. L'utilisatrice saisit l'URL dans ses paramètres et peut installer les mises à jour en un clic. Ses données sont toujours préservées.

Version actuelle de l'app :
Incrémentez la version à chaque mise à jour pour déclencher la notification chez l'utilisatrice.

Comment déployer

1
Cliquez "Télécharger l'app" pour obtenir le fichier HTML à jour
2
Publiez-le sur GitHub (gratuit) : créez un dépôt → uploadez le fichier → copiez l'URL "Raw"
3
Communiquez l'URL Raw à l'utilisatrice — elle la colle dans Config → Mises à jour
4
Pour chaque nouvelle version : répétez l'upload. L'app détecte la nouvelle version et propose l'installation
Contenu à embarquer dans le fichier

Les sites fournisseurs et le PIN concepteur sont toujours inclus. La collection personnelle de votre femme n'est jamais écrasée.

💡 GitHub Pages permet aussi d'héberger l'app directement à une URL permanente (ex: votre-nom.github.io/irisid).
